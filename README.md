# Prognostic Potentials of SMaRT Genes Figure Generation

## Overview
This project generates Kaplan–Meier (KM) survival plots to evaluate the **prognostic potential of BoNE-derived SMaRT genes** across multiple disease conditions.

The figures corresponding to panels **(a–g)** can be regenerated using the notebook:  **generate_figures.ipynb**

---


## Installation

Before running the script, install the required dependency:

```bash
pip install lifelines
```
